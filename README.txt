#PetShop

##Aplicação console que calcula e retorna o melhor PetShop de acordo com a demanda e critérios pré-estabelecidos

###Instruções

Foi utilizado o .NET por motivos de maior familiaridade com a tecnologia.

Assumindo que haja a necessidade da entrada de várias dados de uma só vez, foi projetado para que a aplicação aceito dois métodos de entradas.
Este é o padrão de string que deve ser inserido como entrada
<data> <quantidade de cães pequenos> <quantidade cães grandes>
Exemplo: 03/08/2018 3 5
Ao rodar a aplicação, ela procurará um arquivo com o nome dti.txt em seu disco C ("C:\dti.txt") permitindo
que seja inserido diversos dados de uma única vez afim de consultas múltiplas.
Caso tal arquivo não seja encontrado, será mostrada uma instrução para inserção de uma única string
de dados de forma manual, seguindo o padrão supracitado.
